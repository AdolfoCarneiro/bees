# Roadmap (forward only)

Ordered priorities. No legacy phase numbers.

1. **Frames with real effects** — productivity, mutation, stability, fertility (or subset) tied to apiary math and outputs.
2. **Production pass** — reliable species outputs, comb items, conservative tuning; respect analysis state in UI.
3. **Processing** — centrifuge (or equivalent) separating honey, product, wax; pipe-friendly.
4. **Advanced hive experience** — extensions, clear in-GUI bee presence, automation parity with our own UX.
5. **Species expansion** — first small branch (e.g. managed/cultivated line), new prompts per texture, data + habitat together.
6. **Custom bee entity ADR** — design-only when the above is stable; no rush.

Cross-cutting: prompts under `docs/art/prompts/` for new visuals; `docs/art/asset-prompt-workflow.md` for workflow.

# Curious Bees

Curious Bees is a Minecraft bee genetics mod inspired by the depth of classic Forestry beekeeping, built from scratch for modern vanilla-style bees.

Instead of deterministic A + B = C recipes, bees carry genomes. A baby bee inherits genes from both parents, dominance affects which traits are expressed, and rare mutations can create new species.

## Current Features

- Vanilla-style bee breeding with genetic inheritance
- Dominant and recessive alleles
- Active and inactive traits
- Hybrid and purebred bees
- Early probabilistic species mutations
- Basic analyzer output

## Early Scope

The initial content set is intentionally small. The goal is to prove the genetics loop before adding many species or resource bees.

## Planned

- Tech apiaries and frames
- Expanded species trees
- Data-driven content
- Fabric support later

# Curious Bees

Curious Bees brings Forestry-inspired bee genetics to modern vanilla-style Minecraft bees.

Breed bees, inspect their genomes, stabilize purebred lineages, discover mutations, and gradually build toward deeper tech progression.

## Current Focus

The first versions focus on the genetic breeding loop:

- vanilla-style bee breeding;
- inherited genomes;
- dominant and recessive alleles;
- active and inactive traits;
- probabilistic mutations;
- basic analyzer feedback.

## Not a Resource Bee Mod Yet

Resource bees are planned for the future, but they are not the focus of the first release.

## Status

Early builds are experimental. Back up worlds before testing.

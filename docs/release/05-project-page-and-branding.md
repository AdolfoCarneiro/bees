## 4. Long Description Draft

```md
Curious Bees is a Minecraft mod focused on bee genetics, inheritance, and mutation.

It is inspired by the depth of classic Forestry beekeeping, but it is built from scratch for modern vanilla-style bees.

Instead of deterministic A + B = C recipes, bees carry genomes. A baby bee inherits one allele from each parent, dominance affects which traits are expressed, and rare mutations can create new species.

The initial version starts small on purpose:
- a few wild species,
- a few early mutations,
- basic analyzer feedback,
- and a foundation for future tech progression.

Curious Bees is not a Forestry port and not a Productive Bees fork. It is a new mod built around genetic selection, hybrid potential, and long-term bee breeding progression.
```

## 5. Feature Bullets

Initial feature bullets:

```txt
- Vanilla-style bee breeding with genetic inheritance
- Dominant and recessive alleles
- Active and inactive traits
- Hybrid and purebred bees
- Probabilistic species mutations
- Basic analyzer for bee genetics
- Small starter species set
- Future tech/apiary progression
```

Do not list features that are not implemented yet.

Future features can be listed under "Planned":

```txt
- Tech apiaries and frames
- Data-driven species and mutation definitions
- Expanded biome, Nether, End, and resource bees
- Fabric support
```

## 6. What to Say Clearly

The page should clearly state:

```txt
This is an early build.
Content is intentionally small.
Resource bees are future content.
Back up worlds before testing alpha versions.
```

## 7. What Not to Say

Do not say:

```txt
- Complete Forestry replacement
- All Forestry bees recreated
- Full Productive Bees alternative
- Fabric support available
- Resource bees available
- Stable for all modpacks
```

unless those are actually true.

# Modrinth Release Checklist

## Project Metadata

```txt
- Project name set.
- Summary written.
- Description written.
- License selected.
- Categories selected.
- Icon/logo uploaded.
- Source link added.
- Issue tracker link added.
```

## Version Upload

```txt
- Jar file selected.
- Version name set.
- Version number set.
- Minecraft version selected.
- Loader selected.
- Changelog pasted.
- Dependencies configured.
- Release channel selected: Alpha/Beta/Release.
```

## Review

```txt
- Description does not overpromise.
- Fabric not listed unless supported.
- Resource bees not advertised unless implemented.
- Known issues included for alpha/beta.
- Download artifact matches GitHub release.
```

# CurseForge Release Checklist

## Project Metadata

```txt
- Project name set.
- Summary written.
- Description written in English.
- License selected.
- Class/category selected correctly.
- Additional categories are relevant.
- Logo image uploaded.
- Source link added if available.
```

## File Upload

```txt
- Jar file selected.
- Display name set if useful.
- Release type selected: Alpha/Beta/Release.
- Supported Minecraft version selected.
- Mod loader tag selected.
- Changelog pasted.
- Related projects/dependencies configured.
- Release options checked.
```

## Review

```txt
- Release type is honest.
- Alpha builds are marked Alpha.
- First stable release is not rushed.
- File is not accidentally tagged with wrong loader.
- Known issues are visible.
```

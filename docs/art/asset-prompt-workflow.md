# Asset prompt workflow

## Rule

Before adding a **final** texture, GUI background, block model skin, or icon:

1. Add a prompt document under `docs/art/prompts/` (see [`templates/asset-request-template.md`](templates/asset-request-template.md)).
2. Include target resource path, dimensions, style, palette (hex), usage in-game, acceptance criteria.
3. Integrate the user-provided asset when ready; update [`asset-manifest.md`](asset-manifest.md).

## Dev placeholders

Allowed only to prevent crashes. Mark clearly as **dev placeholder** in filename or adjacent README; they do not complete the art task.
